HTML là gì

HTML viết tắt là của hyper Text Markup language,ngôn ngữ đánh dấu siêu văn bản,là ngôn ngữ đặc biệt dùng để thiết kế nội dung các trang web và dược thể hiện bằng trình duyệt .

Cấu trúc HTML là một cấu trúc dạng cây của các phần tử HTML

CSS là gì

CSS (Cascading Style Sheets) là ngôn ngữ định dạng đặc biệt được dùng đê mô tả cách thể hiện văn bản HTML trong trang web.

Mỗi định dạng CSS đều có dạng chuẩn là bộ chọn {thuộc tính:giá trị;}
